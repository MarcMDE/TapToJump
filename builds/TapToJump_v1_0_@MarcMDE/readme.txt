Tap To Jump v1.0 - Ready To Play

Tap To Jump is a small runner prototype developed by Marc Montagut (MarcMDE)
using C and raylib.

This game (prototype) is still under development, this build (v1.0) could be 
considered and Alpha version.

The game should be updated and available in browsers soon.

The source code can be found in my GitHub: https://github.com/MarcMDE/TapToJump

About the game:
- Press SPACE.
- Reach the end.

The music used in the game is from the League of Legends Warsongs Album:
http://euw.leagueoflegends.com/es/site/warsongs/
